# 当前 Demo 行为基线

记录日期：2026-06-25

## 启动

- `assets/sence/main.scene` 的 Canvas 挂载 `GameBootstrap`。
- 启动链为 `GameBootstrap → SystemManager → WaveSystem / EnemySystem / ViewSystem`。

## 敌人与战斗

- 普通兵、盾兵、骑兵、弓兵均可生成。
- 普通兵、盾兵和骑兵从左向右移动并攻击城门。
- 弓兵到达远程站位后停止，依次执行拉弓、射击和冷却。
- 百步穿杨锁定最靠近城门的敌人，并最多穿透同排后方 1 个敌人。
- 万箭齐发、固若金汤和画地为牢分别执行全体伤害、城门护盾和全体冻结。

## 字袋

- 开局保证一组完整“百步穿杨”。
- 字从字池按补字间隔进入战斗袋。
- 使用成语后，对应字回到可用字池。
- 弃掉的左侧字进入延迟区；当前可用字池耗尽后，弃字才进入下一轮字池。
- 弃字有 3 秒冷却。

## 奖励

- 当前最终行为为五选二。
- 奖励卡可选择、再次点击取消。
- 选满两个后仍需点击确认。
- 确认后两个奖励一次性应用，战斗恢复。

## 失败

- 城门生命归零后停止刷怪、敌人行动和补字。
- 场上敌人被清理。
- 显示失败面板及击杀、等级信息。
- 点击“重新开始”可重置本局。

## 人工回归清单

- [ ] 四类敌人均出现。
- [ ] 弓兵停下并远程射击。
- [ ] 百步穿杨同排穿透，不命中异排敌人。
- [ ] 弃字不会立即抽回。
- [ ] 五选二可取消并确认。
- [ ] 奖励面板期间战斗暂停。
- [ ] 城门归零后可重新开始。
