# v0.7 编辑器系统（Editor System / 工具链版本）

> 目标：将 v0.6 的“规则驱动系统”升级为「可视化编辑 + 内容生产 + 可平衡模拟」完整工具链

---

# 一、v0.7 核心目标

v0.7 不再是玩法升级，而是：

## 🎯 1. 内容生产化（Content Pipeline）
- 所有关卡、成语、Boss、字池全部可编辑

## 🎯 2. 可视化编辑（Editor-first）
- 不改代码即可做新玩法

## 🎯 3. 可模拟平衡（Balance Simulation）
- 在编辑器内跑战斗模拟

---

# 二、整体编辑器架构

```
Unity Editor Tools
 ├── Level Editor（关卡编辑器）
 ├── Idiom Editor（成语编辑器）
 ├── Boss Rule Editor（Boss规则编辑器）
 ├── Letter Pool Editor（字池编辑器）
 ├── Balance Simulator（平衡模拟器）
 ├── Wave Preview Tool（波次预览）
```

# 三、数据驱动核心（统一格式）

## 3.1 关卡定义（LevelData）

```json
{
  "id": "level_01",
  "waves": 10,
  "boss": "ChaosBossRule",
  "letter_bias": {
    "火": 1.5,
    "兵": 1.2
  },
  "enemy_set": [
    "Move+Attack",
    "Move+Disrupt",
    "Move+Support"
  ]
}
```

---

## 3.2 成语定义（IdiomData）

```json
{
  "id": "万箭齐发",
  "pattern": ["万","箭","齐","发"],
  "type": "damage",
  "base_power": 100,
  "scale": 1.15
}
```

---

## 3.3 Boss规则定义（BossRuleData）

```json
{
  "id": "ChaosBoss",
  "events": [
    "OnLetterSpawn",
    "OnWaveStart"
  ],
  "effects": [
    "ShuffleLetterQueue",
    "IncreaseNoise"
  ]
}
```

# 四、Unity Editor结构设计

## 4.1 ScriptableObject数据层

```csharp
[CreateAssetMenu(menuName = "IdiomGame/LevelData")]
public class LevelData : ScriptableObject
{
    public string levelId;
    public int waveCount;
    public List<EnemyConfig> enemies;
    public BossRuleData boss;
}
```

---

## 4.2 成语数据编辑器

```csharp
public class IdiomDataEditor : EditorWindow
{
    private string id;
    private string[] pattern;
    private float basePower;

    void OnGUI()
    {
        id = EditorGUILayout.TextField("Idiom ID", id);
        basePower = EditorGUILayout.FloatField("Base Power", basePower);
    }
}
```

# 五、四大核心编辑器

---

## 5.1 Level Editor（关卡编辑器）

### 功能：
- 配波次
- 配敌人组合
- 配Boss
- 调字池权重

### UI结构：
```
Wave List
 ├── Wave 1
 ├── Wave 2
 ├── Wave 3
Boss Slot
Enemy Presets
Letter Bias Table
```

---

## 5.2 Idiom Editor（成语编辑器）

### 功能：
- 可视化 pattern 拼接
- 数值调节
- 类型分类（damage/control/buff）

### 核心能力：
- 即时Trie验证
- 是否可被当前字池触发

---

## 5.3 Boss Rule Editor（核心创新）

### Boss = 规则图（Rule Graph）

```
OnWaveStart → ShuffleQueue → AddNoise
OnIdiomCast → ReducePower
```

### 节点类型：
- Trigger
- Action
- Condition

---

## 5.4 Letter Pool Editor（字池编辑器）

### 功能：
- 字权重调整
- 污染字开关
- 波次曲线预览
