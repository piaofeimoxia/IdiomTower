# 《成语塔防》游戏方案文档 v1.0

## 一、项目定位

### 1.1 游戏名称

暂定名：

**成语塔防**

备选名称：

* 成语守城
* 一字守长安
* 成语大作战
* 成语封魔录
* 拼成语守城

### 1.2 一句话卖点

**敌军攻城，玩家通过拼出成语释放技能，守住最后一座城门。**

更适合短视频传播的版本：

**别人靠兵守城，我靠成语施法。**

### 1.3 游戏类型

文字合成 + 技能塔防 + 轻肉鸽 + 守城闯关

### 1.4 目标平台

优先考虑：

* 抖音小游戏
* 微信小游戏
* 快手小游戏

### 1.5 核心用户

主要面向：

* 喜欢轻度策略、塔防、合成类小游戏的用户
* 对成语、汉字、国风题材有天然认知的用户
* 喜欢短局、爽感、排行榜、反复挑战的抖音小游戏用户

---

## 二、核心设计思路

《成语施法塔防》的核心不是做“成语答题游戏”，也不是做“文字解谜游戏”，而是把成语变成一种即时战斗资源。

玩家在敌军攻城的压力下，从字池中拖动汉字，拼出成语。每一个成语都会触发符合语义的技能效果。

例如：

* 万 + 箭 + 齐 + 发 = 万箭齐发：全屏箭雨
* 固 + 若 + 金 + 汤 = 固若金汤：城门护盾
* 草 + 木 + 皆 + 兵 = 草木皆兵：召唤士兵
* 火 + 上 + 浇 + 油 = 火上浇油：点燃敌人
* 声 + 东 + 击 + 西 = 声东击西：敌人转向或混乱

玩家不是为了“猜出成语”而玩，而是为了在战斗压力下快速拼出合适的成语，解决当前危机。

---

## 三、核心循环

### 3.1 单局循环

1. 敌军从地图一侧进攻城门。
2. 玩家从底部字池获得随机汉字。
3. 玩家拖动汉字到成语槽中。
4. 拼出完整成语后释放技能。
5. 技能对敌人、城墙、士兵或战场产生效果。
6. 每波结束后获得奖励。
7. 玩家选择一个肉鸽强化。
8. 敌人进入下一波。
9. 守住全部波次则胜利，城门血量归零则失败。

### 3.2 长线循环

1. 通关获得铜钱、军功、成语残卷。
2. 解锁新成语、新技能、新城池、新敌军。
3. 强化某类成语流派。
4. 挑战更高难度关卡。
5. 冲击好友榜、地域榜、全服榜。

## 四、核心玩法

### 4.1 字池系统

每局游戏中，玩家底部会不断刷新汉字。

基础字池示例：

* 万
* 箭
* 齐
* 发
* 火
* 上
* 浇
* 油
* 草
* 木
* 皆
* 兵
* 固
* 若
* 金
* 汤

玩家可以拖动汉字到成语槽位中，组成完整成语。

### 4.2 成语槽

成语槽初始为 4 格。

玩家需要按正确顺序放入汉字。

例如：

```text
万 / 箭 / 齐 / 发
```

拼成后立即释放技能。

后期可以解锁：

* 3 字成语槽
* 4 字成语槽
* 5 字成语槽
* 快捷施法槽
* 自动记忆常用成语槽

### 4.3 成语技能

每个成语对应一个技能。

技能设计原则：

1. 语义和效果要强关联。
2. 玩家看到成语就能理解技能。
3. 技能要有明显视觉表现。
4. 技能效果适合短视频展示。

示例：

| 成语   | 类型   | 技能效果            |
| ---- | ---- | --------------- |
| 万箭齐发 | 伤害   | 对全屏敌人造成箭雨伤害     |
| 火上浇油 | 持续伤害 | 点燃区域内敌人，并提高燃烧伤害 |
| 草木皆兵 | 召唤   | 在城门前召唤一排临时士兵    |
| 固若金汤 | 防御   | 给城门增加护盾         |
| 声东击西 | 控制   | 让一部分敌人转向或短暂混乱   |
| 釜底抽薪 | 削弱   | 降低敌人攻击力和移动速度    |
| 画地为牢 | 控制   | 在地面生成禁锢区域       |
| 破釜沉舟 | 爆发   | 城门血量越低，伤害越高     |
| 风助火势 | 联动   | 增强所有燃烧效果        |
| 背水一战 | 翻盘   | 低血量时全体技能伤害提高    |

## 五、战斗系统

### 5.1 守城目标

玩家需要保护城门。

城门有血量，敌人靠近后会攻击城门。

城门血量归零则失败。

视觉表达：

* 城门裂痕逐渐增加
* 城墙火光和烟雾增强
* 城门血量低时画面震动
* 最后一丝血时出现“危急”提示

### 5.2 敌人类型

基础敌人：

| 敌人 | 特点         |
| -- | ---------- |
| 步兵 | 普通单位，数量多   |
| 盾兵 | 血量高，抗箭雨    |
| 骑兵 | 移动快，容易冲门   |
| 弓兵 | 远程攻击城墙     |
| 工兵 | 专门破坏城门     |
| 术士 | 干扰玩家字池     |
| 将军 | 小 BOSS，高血量 |

### 5.3 BOSS 设计

BOSS 是单局情绪高潮，必须制造“爽—慌—翻盘”的节奏。

示例 BOSS：

#### 黄巾妖道

技能：乱字术
效果：打乱玩家当前成语槽顺序。

#### 破城巨兽

技能：冲城
效果：直线冲撞城门，必须用控制技能打断。

#### 魅惑军师

技能：迷心
效果：让玩家部分字块暂时变成问号。

#### 火攻都尉

技能：火烧城门
效果：城门持续掉血，需要用防御类成语抵消。

## 六、肉鸽强化系统

每波结束后，玩家从 3 个强化中选择 1 个。

强化示例：

### 6.1 成语强化

* 万箭齐发额外多射一轮
* 草木皆兵召唤士兵数量 +2
* 固若金汤护盾提升 30%
* 火上浇油燃烧持续时间 +2 秒

### 6.2 流派强化

* 火系成语伤害 +20%
* 防御成语冷却降低
* 召唤单位血量提升
* 控制类成语持续时间增加

### 6.3 字池强化

* 字池刷新速度提升
* 出现关键字概率提高
* 可锁定 1 个字不被刷新
* 成语槽错误放置可撤回一次

## 七、流派设计

### 7.1 箭雨流

核心成语：

* 万箭齐发
* 百步穿杨
* 箭无虚发

特点：

持续远程输出，适合清杂兵。

### 7.2 火攻流

核心成语：

* 火上浇油
* 风助火势
* 烈火燎原

特点：

范围燃烧，适合打密集敌人。

### 7.3 防御流

核心成语：

* 固若金汤
* 铜墙铁壁
* 安如泰山

特点：

强化城门，适合持久战。

### 7.4 召唤流

核心成语：

* 草木皆兵
* 千军万马
* 兵临城下

特点：

召唤士兵堵路，适合拦截快速敌人。

### 7.5 控制流

核心成语：

* 画地为牢
* 声东击西
* 围魏救赵

特点：

控制敌人走位和节奏，适合高手玩家。

## 八、差异化设计

### 8.1 和普通成语游戏的区别

普通成语游戏通常是答题、填字、消除。

本项目的区别是：

**成语不是答案，而是技能。**

玩家拼成语不是为了过文字关，而是为了守城、杀敌、翻盘。

### 8.2 和《赵云与阿斗》的区别

《赵云与阿斗》的核心是文字合成单位。

本项目的核心是：

**拼成语释放技能。**

差异点：

* 不是合成武将，而是拼出技能。
* 不是保护阿斗，而是守城门。
* 不是三国名将认知，而是成语语义认知。
* 不是单位羁绊，而是成语流派和技能连携。

## 九、短视频传播设计

### 9.1 15 秒素材结构

#### 素材一：极限翻盘

0-3 秒：城门只剩 1 点血，敌军压城。
3-7 秒：玩家快速拖字，拼出“万箭齐发”。
7-10 秒：全屏箭雨落下，敌军被清空。
10-15 秒：出现文案：“这游戏，成语真的能救命。”

#### 素材二：成语联动

0-4 秒：敌军密集冲来。
4-8 秒：玩家拼出“火上浇油”。
8-11 秒：再拼“风助火势”。
11-15 秒：火焰扩大，全场燃烧。

#### 素材三：守城危机

0-3 秒：BOSS 冲向城门。
3-7 秒：玩家拼“画地为牢”。
7-11 秒：BOSS 被困住。
11-15 秒：玩家拼“万箭齐发”完成击杀。

### 9.2 买量标题方向

* 城门只剩 1 血，我用一个成语翻盘了
* 原来成语真的能打仗
* 别人守城靠兵，我守城靠成语
* 万箭齐发到底有多爽？
* 小学生都会的成语，在这里是大招
* 拼错一个字，城门就没了

## 十二、关键风险

### 12.1 成语识别门槛过高

风险：

部分玩家不会成语，可能觉得难。

解决方案：

* 前期只用高认知成语。
* 提供半提示机制。
* 成语槽显示灰色影子提示。
* 前几关自动推荐可拼成语。

### 12.2 操作压力过大

风险：

敌人进攻时还要拼字，可能手忙脚乱。

解决方案：

* 前期节奏慢。
* 成语槽支持撤回。
* 关键字可锁定。
* 低难度提供自动排序辅助。

### 12.3 短视频看不懂

风险：

如果画面只是大量汉字，观众可能不知道发生了什么。

解决方案：

* 技能表现必须夸张。
* 成语释放时大字特效突出。
* 城门血量和危机感必须明显。
* 素材要围绕“1 血翻盘”“全屏清怪”“BOSS 被控”做。

## 十三、核心体验目标

玩家在单局中应该体验到：

1. 拼出成语的理解爽感。
2. 释放技能的视觉爽感。
3. 敌军压城的紧张感。
4. 极限翻盘的成就感。
5. 解锁新成语的收集欲。
6. 排行榜超过别人的胜负欲。

---

## 十四、总结

《成语施法塔防》是一款以成语为技能核心的轻度塔防小游戏。

它的核心优势在于：

* 成语有天然文化认知。
* 拼字过程有操作参与感。
* 成语技能有强语义表达。
* 守城塔防有明确压力目标。
* 极限翻盘适合短视频传播。
* 轻肉鸽强化能支撑复玩。

项目核心不是“考成语”，而是：

**用成语打仗。**

最终目标是做出一款用户刷到视频能立刻看懂、点进去能快速上手、失败后想再来一局的抖音小游戏。
